Creating a Local Database

Local Environment

Unlike working in a shared or remote environment, a local development environment provides much more flexibility in your setup and control. However, if your web applications are going to work in both locations, both need to be set up the same. This is very important!

Build the remote system first!

Prior to building the database locally do the remote system first. This is because the remote system will have specific requirements that must be met. If you meet those requirements it is fairly easy to replicate them locally. If you have not already done so, view the remote database setup video.
