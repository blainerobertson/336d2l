Ch. 13 Enhancement

Purpose

Each enhancement provides an opportunity to solve problems and apply the concepts covered in the current and previous topics.

Assumptions

This enhancement builds upon chapter 12. The enhancement from chapter 12 is the starting point for this enhancement.

Make a Copy

You must have completed enhancement 12 so that you are familiar with the Future Value application and how it operates. Make a copy of your chapter 12 enhancement folder and rename the copy to reflect that it is the chapter 13 enhancement. Do all work in the chapter 13 enhancement location.

Tasks

You will rework the Future Value application by adding 3 custom functions to the application and applying them to the operation of the controller. The specific functions and what they do are described below:

  1. Create a functions file in the folder where all three functions will be created, be sure to bring this file into scope within the controller to use the functions.
  2. Create a function to calculate the future value. This function should provide for compounding the interest on either a monthly or yearly basis. To do that, you can include a parameter for compounding the interest monthly that has a default value of FALSE.
  3. Create a second function to apply currency formatting to a value and use it with values before sending them to the view to be displayed.
  4. Create a third function to apply percent formatting to a value and use it with percentage values before sending them to the view to be displayed.

Submission

  • Build and test your code in the local development environment.
  • Check your work to insure that it is operating correctly.
  • When satisfied that the code is operational, upload the enhancement folder to the remote production server.
  • Create a link from your exercises page to the index page in the enhancement folder and test that it works.
  • Fully test the enhancement application for operation on the production server. Correct any errors that are found.
  • When the enhancement is finished, tested and working, submit the controller and view to the code submission dropbox using the directions found there.

Grading Matrix

  • The functions file exists: 2 points in objective 2
  • The functions file is “in scope” of the controller: 3 points in objective 3
  • The function to calculate future values works with either yearly or monthly compounding of interest and performs calculations correctly: 15 points in objective 2
  • The currency formatting function works as required: 5 points in objective 2
  • The percent formatting function works as required: 5 points in objective 2

Enhancement Value

  • Objective 2: 27 points
  • Objective 3: 3 points
