Dynamic Project FAQ

The Dynamic Project is where you decide how you will demonstrate your mastery of the concepts, skills, and techniques learned in this course. Use this Discussion Board to post questions and insights pertaining to the project in order to help one another plan, build, and complete the Dynamic Project.

Event Timeline

Today - Begin thinking about a project that you would like to do, that is web based, that you could build for the dynamic project. Right now it will probably be vague, but as the course progresses, begin to firm up your ideas. Review the dynamic project requirements now so you can begin thinking about it.

Week 08 - Your project proposal is due as a word processing document. You will submit your proposal using the I-Learn submission tool to your teacher. The criteria for the proposal should be carefully reviewed to make sure your proposal is accurate and complete.

Week 13 - Two peer reviews should be done by other members of the class in order to catch errors and provide you feedback prior to the project being due.

Week 14 - The dynamic project is due for teacher assessment. Make sure your project meets the criteria by carefully reviewing the grading matrix in advance.
