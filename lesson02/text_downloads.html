Download Text Materials

DOWNLOAD

  1. Download the chapter and exercise files for the text.
  2. Unzip the downloaded file.

Contents

The unzipped folder should be named "murach_files".

Open the folder. There should be 3 sub-folders:

  • book_apps — Contains the files and code found in the book chapters.
  • ex_starts — Contains the starting files for the end of chapter exercises.
  • ex_solutions — Contains the solutions to the end of chapter exercises.

Copy to htdocs

In setting up the tools for the course you should have an "htdocs" folder for your local development environment. This local environment is the perfect location to place files in order to do the chapter work, exercises and enhancements.

As needed, copy the individual chapter or exercise folder into the homework folder on your machine. Then open NetBeans and begin working.

Testing

Be sure to test your work in the production server as well. To do so, upload the files to your remote host and test it live in your browser.
